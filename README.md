# 仙山我做主

一款以仙山为背景的葫芦种植培育游戏。在灵山之上开辟灵田，种植培育各种拥有独特能力的仙葫芦，感受修仙种田的乐趣。

## 游戏简介

你是一位初入仙山的修仙者，获得了一片灵田和几颗翠玉葫芦种子。通过种植、培育、收获各种葫芦，积累灵石资源，解锁更多灵田和稀有葫芦品种，最终成为仙山之主。

## 核心玩法

### 种植葫芦

- **购买种子**：在右侧「仙市」面板购买各种葫芦种子
- **种植方式**：
  - 点击种子选中，再点击空灵田种植
  - 直接将种子拖拽到空灵田上
  - 右键空灵田打开种植菜单
- **生长阶段**：种子 → 发芽 → 藤蔓 → 开花 → 结果 → 成熟（共6个阶段）
- **收获**：点击成熟的葫芦即可收获，获得灵石和种子

### 日常管理

- **浇水**：点击灵田浇水，浇水后生长速度提升（雨天自动浇水）
- **施肥**：使用仙灵肥加速生长30%
- **下一天**：点击「下一天」推进时间，葫芦会生长，天气会变化
- **批量操作**：使用「浇水」和「收获」按钮一键操作所有灵田

### 灵田开垦

- 初始拥有12块灵田，共30块可开垦
- 点击锁定地块花费灵石开垦
- 开垦费用随已开垦数量递增

## 天气系统

游戏拥有8种天气，每天自动变化，不同天气对葫芦生长有显著影响。

| 天气 | 图标 | 特点 |
|------|------|------|
| 晴朗 | ☀️ | 阳光明媚，适合火系、金系葫芦 |
| 多云 | ⛅ | 气候温和，适合土系葫芦 |
| 雨天 | 🌧️ | 自动浇水，适合水系、木系葫芦 |
| 雷暴 | ⛈️ | 风雨交加，适合雷系、风系葫芦 |
| 大风 | 💨 | 狂风呼啸，适合风系葫芦 |
| 雪天 | ❄️ | 天寒地冻，适合冰系葫芦 |
| 迷雾 | 🌫️ | 灵气充盈，适合混沌、木系葫芦 |
| 酷暑 | 🔥 | 烈日炎炎，适合火系、金系葫芦 |

### 天气喜好

每种葫芦都有喜欢的天气和厌恶的天气：

- **喜欢**的天气下生长速度提升 30%~100%
- **厌恶**的天气下生长速度降低 40%~60%
- 头部信息栏会显示当前天气和明日天气预报，方便规划种植

### 场景天气效果

- 雨天/雷暴：画面出现雨滴粒子，雷暴时有闪电
- 雪天：飘落雪花，草地和树木逐渐积雪
- 迷雾：半透明雾层飘动
- 大风：风纹划过画面，树木和草叶大幅摇摆
- 酷暑：热浪波纹，草地和树叶变黄
- 晴朗：右上角太阳光芒

## 2.5D 场景

游戏采用 Canvas 2.5D 等距视角渲染，包含丰富的场景元素：

- **灵田**：5×6 等距网格，带3D深度效果
- **葫芦生长**：6个阶段各有独特的2.5D图形，成熟时带元素特效和光晕
- **茅庐**：茅草屋顶带质感，烟囱冒烟，窗户透出暖光
- **篱笆**：环绕灵田的木栅栏
- **树木**：松树和圆树分布在围栏外，随风摇摆
- **草皮**：围栏外草地随风摆动
- **仙鹤/仙鹿**：随机出现的灵兽，在灵田周围漫步

## 操作说明

| 操作 | 效果 |
|------|------|
| 左键点击空灵田 | 种植已选中的种子 |
| 左键点击葫芦 | 浇水 / 收获（成熟时） |
| 右键点击葫芦 | 查看详情 |
| 右键点击空灵田 | 打开种植菜单 |
| 拖拽种子到灵田 | 直接种植 |
| 点击「下一天」 | 推进游戏时间 |

## 技术实现

- **前端**：HTML5 Canvas + CSS3 + JavaScript
- **渲染**：Canvas 2.5D 等距视角渲染引擎
- **动画**：requestAnimationFrame 60fps 动画循环
- **数据持久化**：LocalStorage 自动存档
- **天气概率**：马尔可夫链概率转移模型
